# Description

Full Stack application,NodeJs + Typescript para API, Jest para testes de int e unit, Prisma ORM para PostgreSQL + Redis para Cache</br>

</br>
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
